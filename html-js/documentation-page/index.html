Conditionals

alright the conditionals...

If (cactus === 'best things ever') { console.log('cactus is best things ever'); }else { console.log('cactus is not best things ever :('); }

you can use a if else statement. Which will do anything in the {} as long as the condition in () is equal to true. And the Else is for what to do when the condition is equal to False. You don't need else statement if you don't need it.

Promises

Pretty easy. If you have a function or some kind of action, that could take some unknown time and that could throw an error, then you should use a promise. You can declare a promise by using the new operator. Use resolve or any call sign in the func args.

const mypromise = new Promise((resolve, reject) => { setTimeout(()=> { resolve('foo'); }, 300); });

You can use await or .then(), .catch(). Await can only work in async function... Then is what will be run, when the promise is resolved. Catch is used to handle errors... Be carefull, you need to use function in the .then or .catch

mypromise.then((resolveValue)=> { console.log(resolveValue); })
